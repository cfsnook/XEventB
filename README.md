XEvent-B Feature
================
This feature provides the eXtended Event-B (XEvent-B).

IMPORTANT:
---------
- Require XText from the update site
  http://download.eclipse.org/modeling/tmf/xtext/updates/composite/releases/

KNOWN ISSUES:
------------
- None at the moment
